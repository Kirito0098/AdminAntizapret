# 🤖 Telegram в AdminAntizapret

**AdminAntizapret** поддерживает вход через Telegram и запуск **Telegram Mini App**. Через эту интеграцию можно авторизоваться в панели без ручного ввода логина и пароля, а также открывать адаптированную версию панели прямо внутри Telegram.

---

## 📝 Что включает интеграция

- вход в панель через Telegram Login Widget;
- привязку Telegram ID к пользователю панели;
- запуск Telegram Mini App из бота;
- отображение Telegram-пользователя внутри Mini App;
- защиту от некорректных или устаревших Mini App-сессий.

---

## 📋 Что нужно заранее

Перед настройкой подготовьте:

- доступ к панели под ролью `admin`;
- Telegram-бота, созданного через BotFather;
- публичный HTTPS-адрес панели;
- валидный SSL-сертификат от доверенного центра сертификации.

---

## ⚠️ Важно для Telegram Mini App

Telegram Mini App не рассчитан на запуск через обычный HTTP и не работает с самоподписанными сертификатами. Для нормальной работы нужен публичный HTTPS.

| Вариант публикации | Статус | Комментарий |
| --- | --- | --- |
| `http://IP:5050` | Не поддерживается | Mini App не откроется |
| `https://IP:5050` с самоподписанным сертификатом | Обычно не работает | Telegram WebView не доверяет такому сертификату |
| `https://домен:5050` с валидным сертификатом | Поддерживается | Подходит для прямой публикации приложения |
| `https://домен` через Nginx reverse proxy + Let's Encrypt | Поддерживается | Рекомендуемый вариант |

## 🚀 Порядок настройки

Настройка Telegram в AdminAntizapret обычно выглядит так:

1. Создать Telegram-бота через BotFather.
2. Включить Telegram-авторизацию в панели.
3. Привязать Telegram ID к нужному пользователю.
4. Проверить вход через Telegram.
5. Настроить кнопку запуска Telegram Mini App.

---

## 1. Создайте бота в BotFather

1. Откройте BotFather в Telegram.
2. Выполните команду `/newbot`.
3. Задайте имя бота.
4. Задайте `username` бота.
5. Сохраните токен, который выдаст BotFather.

Пример токена:

```text
123456789:AAExampleExampleExampleExampleExample
```

> **Важно:** не передавайте токен третьим лицам. Если токен был скомпрометирован, перевыпустите его через BotFather.

---

## 2. Включите Telegram-авторизацию в панели

Откройте в панели:

`Настройки` → `Telegram авторизация`

Заполните поля:

- `Username бота Telegram`;
- `Токен бота Telegram`;
- `Макс. возраст Telegram авторизации`.

Рекомендуемое значение max age:

```text
300 секунд
```

После заполнения:

1. Нажмите `Сохранить Telegram авторизацию`.
2. Убедитесь, что статус Telegram-логина стал активным.

### Проверка форматов

В проекте проверяются следующие форматы:

- `Telegram ID` — только цифры, длина от 5 до 21 символа, без ведущего нуля;
- `Username бота` — от 5 до 64 символов, латиница, цифры и `_`;
- `Токен бота` — число, двоеточие и строка токена.

---

## 3. Привяжите Telegram ID к пользователю панели

Откройте:

`Настройки` → `Пользователи и доступ`

Дальше:

1. Выберите существующего пользователя или создайте нового.
2. Найдите поле `Telegram ID`.
3. Укажите числовой Telegram ID пользователя.
4. Сохраните изменения.

Что важно учитывать:

- один Telegram ID нельзя привязать к двум пользователям панели;
- если сохранить пустое значение, Telegram-привязка будет удалена;
- привязку можно задать как для `admin`, так и для `viewer`, если это допускает ваш сценарий доступа.

---

## 4. Проверьте вход через Telegram

После настройки привязки пользователь может войти в панель через Telegram.

### Вариант A. Вход через страницу логина

1. Откройте страницу входа в панель.
2. Нажмите кнопку входа через Telegram.
3. Подтвердите вход в Telegram.
4. После успешной проверки пользователь будет авторизован в панели.

Если Telegram ID привязан корректно, вход выполнится именно в соответствующего пользователя AdminAntizapret.

### Вариант B. Вход через Telegram Mini App

1. Откройте `Настройки` → `Telegram авторизация`.
2. Скопируйте ссылку из блока Mini App.
3. Добавьте её в бота как Web App-кнопку.
4. Откройте приложение из Telegram.

---

## 5. Настройте запуск Telegram Mini App

Самый удобный способ запуска Mini App — назначить его на кнопку меню бота.

Через BotFather:

1. Выполните команду `/setmenubutton`.
2. Выберите нужного бота.
3. Укажите тип кнопки `Web App`.
4. Вставьте ссылку Mini App из панели.

После этого кнопка `Menu` в чате с ботом будет открывать Mini App напрямую.

---

## ✅ Что проверить после настройки

После завершения настройки рекомендуется проверить весь сценарий:

1. Выйдите из панели.
2. Выполните вход через Telegram на странице логина.
3. Откройте Mini App через бота.
4. Проверьте, что в интерфейсе отображается правильный пользователь панели.
5. Проверьте Telegram username.
6. Проверьте Telegram ID.

Если данные совпадают, интеграция настроена корректно.

## ❗ Частые проблемы

### Telegram-логин не включается

Причина: не указан `username` бота или токен.

Решение:

1. Откройте `Настройки` → `Telegram авторизация`.
2. Заполните оба поля.
3. Сохраните настройки повторно.

### Telegram-аккаунт не привязан к пользователю панели

Причина: Telegram ID отсутствует в карточке пользователя.

Решение:

1. Откройте `Настройки` → `Пользователи и доступ`.
2. Найдите нужного пользователя.
3. Укажите его Telegram ID.
4. Сохраните изменения.

### На устройстве несколько Telegram-аккаунтов, открывается не тот пользователь

Причина: Mini App открыт из другого Telegram-аккаунта или использует старую сессию.

Решение:

1. Закройте Mini App.
2. Переключитесь на нужный Telegram-аккаунт.
3. Откройте Mini App заново из бота.
4. Сверьте Telegram ID в интерфейсе.

### Mini App не открывается

Причина: используется HTTP, самоподписанный сертификат или запуск идёт вне Telegram.

Решение:

1. Используйте публичный HTTPS.
2. Убедитесь, что сертификат валиден.
3. Запускайте Mini App только из Telegram-клиента.

---

## 📘 Связанные разделы панели

- `Настройки` → `Telegram авторизация` — включение Telegram-входа и получение ссылки Mini App;
- `Настройки` → `Пользователи и доступ` — привязка Telegram ID к пользователям;
- `Вход` — проверка Telegram Login Widget;
- `Telegram Mini App` — быстрый доступ к панели из Telegram.

Если вы настраиваете Telegram впервые, сначала включите авторизацию в панели, затем привяжите Telegram ID, и только после этого переходите к Mini App.
