# Инструкция по добавлению Telegram в AdminAntizapret

Этот файл описывает полный путь: как включить вход через Telegram, как привязать Telegram ID к пользователю панели и как запустить Telegram Mini App.

## Что нужно заранее

- Доступ к панели под ролью admin.
- Рабочий Telegram-бот, созданный через BotFather.
- Публичный HTTPS-адрес панели (для Mini App это важно).

## Важное предупреждение про HTTP и сертификаты

1. Telegram Mini App не рассчитан на запуск с HTTP-ссылок.
2. HTTPS с самоподписанным сертификатом обычно не подходит: Telegram WebView не доверяет такому сертификату.
3. Для стабильной работы нужен публичный HTTPS с сертификатом от доверенного центра сертификации (например, Let's Encrypt).
4. В рамках AdminAntizapret Mini App будет работать только при одном из вариантов:
 1) Использовать собственный домен и получить сертификаты Let's Encrypt.
 2) Использовать Nginx как reverse proxy с сертификатами Let's Encrypt.

## Шаг 1. Создайте бота в BotFather

1. Откройте BotFather в Telegram.
2. Выполните команду /newbot.
3. Сохраните username бота (например: MyPanelAuthBot).
4. Сохраните токен бота (формат вида 123456789:xxxxxxxxxxxxxxxxxxxx).

Важно: никому не передавайте токен. Если токен утек, перевыпустите его через BotFather.

## Шаг 2. Включите Telegram авторизацию в панели

1. Откройте Настройки -> Telegram авторизация.
2. В блоке Вход в панель через Telegram заполните username бота Telegram.
3. Введите токен бота Telegram.
4. Задайте Макс. возраст Telegram авторизации (рекомендуется 300 секунд).
5. Нажмите Сохранить Telegram авторизацию.
6. Убедитесь, что статус Telegram логина стал включен.

Проверка форматов в проекте:

1. Telegram ID: только цифры, 5..21 символ, не начинается с 0.
2. Username бота: 5..64 символа, латиница, цифры, underscore.
3. Токен: число, двоеточие, затем длинная строка символов.

## Шаг 3. Привяжите Telegram ID к пользователю

1. Откройте Настройки -> Пользователи и доступ.
2. Выберите нужного пользователя.
3. В секции Telegram ID укажите numeric Telegram ID.
4. Нажмите Сохранить Telegram ID.

Можно также указать Telegram ID при создании нового пользователя.

Примечания:

1. Один Telegram ID нельзя привязать к двум разным пользователям.
2. Пустое поле Telegram ID и сохранение = отвязка Telegram от пользователя.

## Шаг 4. Настройте вход пользователя через Telegram

Вариант A (веб-вход):

1. Откройте страницу логина панели.
2. Нажмите вход через Telegram.
3. После подтверждения вход выполнится в пользователя, к которому привязан этот Telegram ID.

Вариант B (Mini App):

1. Откройте Настройки -> Telegram авторизация.
2. Скопируйте ссылку из блока Ссылка на Mini App.
3. Добавьте эту ссылку в бота (кнопка меню или web_app кнопка).
4. Открывайте Mini App только из Telegram.

## Шаг 5. Рекомендуемая настройка меню бота для Mini App

Через BotFather:

1. /setmenubutton.
2. Выберите бота.
3. Тип кнопки: Web App.
4. URL: вставьте ссылку Mini App из панели.

После этого кнопка Menu в чате с ботом будет открывать Mini App.

## Проверка после настройки

1. Выйдите из панели.
2. Выполните вход через Telegram.
3. Откройте Mini App из бота.
4. В шапке Mini App проверьте панельного пользователя.
5. В шапке Mini App проверьте Telegram username.
6. В шапке Mini App проверьте Telegram ID.

Если данные совпадают, привязка сделана корректно.

## Частые проблемы и решения

Проблема: Telegram логин выключен.

1. Причина: пустой токен или username бота.
2. Решение: заполните оба поля в Настройки -> Telegram авторизация и сохраните.

Проблема: Этот Telegram аккаунт не привязан ни к одному пользователю панели.

1. Причина: Telegram ID не записан в карточке пользователя.
2. Решение: добавьте Telegram ID в Настройки -> Пользователи и доступ.

Проблема: На одном телефоне 2 Telegram-аккаунта, открывается не тот пользователь.

1. Статус: в проекте добавлена защита от stale-сессии Mini App и показ текущего TG ID в шапке.
2. Решение: закрыть Mini App и открыть заново из нужного аккаунта; сверить TG ID в шапке.

Проблема: Mini App не открывается.

1. Причина: ссылка на HTTP, самоподписанный сертификат HTTPS или запуск не из Telegram.
2. Решение: используйте публичный HTTPS с валидным сертификатом (например, Let's Encrypt) и открывайте Mini App только из Telegram клиента.
