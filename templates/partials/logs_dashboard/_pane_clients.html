Подключенные клиенты

Иногда у подключения видно "Устройство: Не определено" и "Версия: Не определено". Это нормально: эти данные приходят только от самого клиентского приложения во время подключения. Если клиент не передал их или давно не переподключался, панель покажет "Не определено". {% if openvpn_log_tail_lines == 0 %} Сначала панель читает текущие данные OpenVPN management (без лимита по строкам). Если свежих данных нет, используются записи из БД за последние 7 дней при совпадении имени клиента и IP. {% else %} Сначала панель читает хвост OpenVPN management: последние {{ openvpn_log_tail_lines }} строк. Если свежих данных нет, используются записи из БД за последние 7 дней при совпадении имени клиента и IP. {% endif %}

Как определяется "возможно зависшая": если у одного пользователя есть несколько сессий с одним и тем же IP в одном профиле, более старая сессия помечается как "возможно зависшая", а самая новая считается активной.
