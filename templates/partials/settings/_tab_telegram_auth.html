Как добавить через BotFather

  1. Откройте @BotFather в Telegram и выберите вашего бота.
  2. Откройте Bot Settings -> Menu Button (или используйте команду /setmenubutton).
  3. Выберите бота, вставьте ссылку запуска Mini App из поля выше и сохраните.
  4. Откройте чат с ботом и проверьте кнопку Menu/Launch.

Важно для Telegram Mini App

Mini App будет работать только при одном из вариантов:

  • Свой домен + HTTPS-сертификат Let's Encrypt.
  • Nginx reverse proxy + HTTPS-сертификат Let's Encrypt.
