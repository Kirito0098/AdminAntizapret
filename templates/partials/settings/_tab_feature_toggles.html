Управляйте фоновыми задачами и целыми разделами панели. Отключайте неиспользуемое, чтобы снизить нагрузку на сервер — изменения сохраняются в .env и применяются сразу.

Оценки нагрузки приблизительные — зависят от числа клиентов, активности админов и расписания cron.

По умолчанию всё включено. Фоновые задачи можно отключать без скрытия интерфейса — разделы UI останутся на месте.

Разделы приложения при отключении скрываются из меню; их страницы и API возвращают ошибку «модуль отключён».
